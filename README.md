# Assignment 5

## Overview

For assignment 5 i added basic JavaScript code in order to create a more responsive and interactive webpage for users. The goal was to create a simple yet effective user interaction using Javascript.

## What the interaction does

I have created two popup interactions on my website:

* A **welcome alert message** appears when the page loads, greeting users when they visit the website.
* A **"Contact Me" popup** appears when the user clicks the “Contact Me” button, that displays my email address.

## What triggers the interaction

* The welcome message is triggered automatically when the page opens/loads.
* The contact message is triggered by a **click event** on the “Contact Me” button that works with JavaScript functionality.

## How it works

I created a JavaScript function called `showContact()` that contains the alert message which displays my email address, then used an **event listener** to listen/detect a button click and trigger the function to pop up when the user interacts with the page.

## Why I chose this interaction

I chose this interaction because it's not only basic/simple but it allowed me to slowly integrate these JavaScript concepts and functions into my webpage while still keeping the relevancy of it. It also allows users to quickly access my contact details if needed while also demonstrating my understanding of basic JavaScript concepts.
